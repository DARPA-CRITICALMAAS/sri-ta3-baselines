### Install
1. Create the conda environment - `conda create -n ta3-baseline python=3.8`
2. Activate conda environment - `conda activate ta3-baseline`
3. Install tool libraries - `pip3 install -r requirements.txt`
4. Download the Lawley'22 datacube titled "2021_Table04_Datacube.csv" and place it within the `data` folder.

### MVT Expirments
All MVT experiments are within a corresponding notebook. Please open and view/run the cells of the notebook to view/run the experiment.
